Basic PanelSubtitle

A panel is basic UI element that you are probably going to use a lot. It is necessary for displaying the content in a compelling form. Panels can contain tables, plain text, lists and almost all types of content you want to display.

Panels are great for widgets. They can be combined with various tools like icons, dropdowns, buttons, form elements and tabs. They could be closable, collapsible and maximizable.
